IVariable メソッド

IVariable 型は下記のメンバーを公開します。

メソッド
  名前説明
Exists
変数やリストが存在するかどうかを取得します。
Get(String)
KeyToKey側の変数を取得します。
GetT(String)
KeyToKey側の変数を指定した型にキャストして取得します。
GetList
リストを取得します。
Set
KeyToKey側の変数に値を代入します。 変数が存在しない場合は変数を作成します。
SetGlobal
KeyToKey側のグローバル変数に値を代入します。 変数が存在しない場合は変数を作成します。
SetList
リストをセットします。
SetLocal
KeyToKey側のローカル変数に値を代入します。 変数が存在しない場合は変数を作成します。
